# Auction Management System

## Project Overview
This project is an Auction Management System developed to manage auctions, items, bidders, and related transactions.
It focuses on applying database concepts using SQL and Oracle Forms through a practical, form-based application.

The system demonstrates proper database design, data integrity, and structured interaction between the database and user interfaces.

---

## Technologies Used
- Oracle Database
- SQL (DDL & DML)
- Oracle Forms
- ERD & Relational Database Design
